# ShieldHold Portfolio

Professional Instagram creator management website built with Next.js.

## Features
- Modern responsive design
- Smooth animations
- Services showcase
- Pricing plans
- Contact information

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000
